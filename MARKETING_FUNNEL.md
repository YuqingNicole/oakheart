## 目标客户画像

### Persona 1: 独立设计师 (First-Timer)
```
特征:
• 第一次制造桌游
• 预算有限 (通常 $5K-$20K)
• 需要大量指导
• 可能走众筹或自费出版

痛点:
• 不知道如何选择制造商
• 担心被坑 / 质量问题
• 不清楚真实成本和时间
• 沟通困难 (语言/时区)

决策因素:
• 透明的定价
• 耐心的沟通
• 小批量起订
• 成功案例参考
```

### Persona 2: 众筹出版商 (Crowdfunder)
```
特征:
• 正在或即将进行众筹
• 中等预算 ($20K-$100K)
• 时间敏感 (承诺交付日期)
• 有一定经验但需要支持

痛点:
• 准确的成本估算 (众筹定价)
• 交付时间可靠性
• 质量一致性
• 物流和履约

决策因素:
• 准时交付记录
• 灵活的付款条款
• 物流解决方案
• 快速响应能力
```

### Persona 3: 成熟出版商 (Established Publisher)
```
特征:
• 已有多款游戏出版经验
• 大预算 ($100K+)
• 寻找长期合作伙伴
• 关注效率和成本优化

痛点:
• 产能和稳定性
• 成本优化空间
• 供应链风险分散
• 质量管理体系

决策因素:
• 产能规模
• 成本竞争力
• 质量认证
• 长期合作条款
```
